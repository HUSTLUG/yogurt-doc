## 收录架构

## 使用说明

新建或修改 `/etc/portage/repos.conf`：

```shell
[DEFAULT]
main-repo = gentoo

[gentoo]
location = /usr/portage
sync-type = rsync
sync-uri = rsync://211.67.25.90:8081/gentoo-portage/
auto-sync = yes
```
