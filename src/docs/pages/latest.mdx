# 最新代码配置

## 1. 代码拉取

首先需要安装pip3, 通过pip3 再安装mercurial
```shell
sudo yum install epel-release -y
sudo yum install python3-devel python3 -y
sudo yum groupinstall 'development tools' -y
pip3 --version
pip3 install mercurial
``` 

注意我们的python版本需要为python3.7+
```shell
yum install python38
```
